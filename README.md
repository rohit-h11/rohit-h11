# 💫 About Me:
Hey, I'm Rohit — a first-year Energy Engineering student at IIT Guwahati who spends most of his time writing code rather than studying energy.<br>I build full-stack web apps, mostly with Next.js, Node.js, TypeScript, and MongoDB. Currently working on two live projects at IIT Guwahati — CCW Contests (a competitive programming platform for the Coding Club) and Habit IITG (the institute's mess management app used by the student community).<br>I also do competitive programming on Codeforces and dabble in game development — I've shipped both a 2D platformer and contributed to a 3D game at Kriti, IIT Guwahati's techfest.
